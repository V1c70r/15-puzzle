# 15 puzzle

## Requirements

* node.js 14

## Installation

`npm run setup`

## Run

`npm run start` - console version

`npm run start:svelte` - browser version on port 5000

or

open [15 puzzle](https://v1c70r.github.io/15-puzzle/) in a browser
